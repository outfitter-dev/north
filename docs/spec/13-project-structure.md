## Project Structure

```
.north/                         # Project config + assets (committed)
├── config.yaml                 # Main config, dials, extends declaration
├── presets/                    # In-repo presets (committed)
├── rules/
│   ├── no-raw-palette.yaml
│   ├── no-arbitrary-values.yaml
│   └── no-inline-color.yaml
├── tokens/
│   ├── base.css                # Hand-authored: extends shadcn, adds scales
│   ├── effects.css             # Hand-authored: composite tokens
│   └── generated.css           # Generated by `north gen` (committed)
├── state/                      # Generated state (ignored)
│   └── index.db                # SQLite index (default)
└── .gitignore                  # Ignores state/ by default

src/                            # Application code (outside .north)
└── components/
    └── composed/               # Project-specific composed components
```

**Directory convention:**
- `.north/` — source-of-truth config/presets/tokens/rules (committed)
- `.north/state/` — derived artifacts and cache (ignored)
