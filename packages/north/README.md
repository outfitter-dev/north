# @outfitter/north

Design system enforcement CLI tool for maintaining consistency across your codebase.

## Installation

### Global Installation

```bash
bun add -g @outfitter/north
```

### Per-Project Installation

```bash
bun add -D @outfitter/north
```

## Usage

```bash
# Check version
north --version

# Get help
north --help
```

## Development

This package is part of the North monorepo.

### Building

```bash
bun run build
```

### Type Checking

```bash
bun run typecheck
```

### Linting

```bash
bun run lint
```
